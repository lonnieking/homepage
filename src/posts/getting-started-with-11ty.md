11ty (Eleventy) is a powerful static site generator that makes it easy to build fast, modern websites.

## Why 11ty?

11ty is flexible, has zero config by default, and supports multiple template languages. Whether you're building a blog, portfolio, or documentation site, 11ty can handle it.

## Getting Started

To get started with 11ty, you'll need Node.js installed. Then simply:

```bash
npm install --save-dev @11ty/eleventy
```

And create your first template file. That's it!
